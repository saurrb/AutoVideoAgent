# Page 4 Relationship Setup (AI-Only)

This page no longer uses Excel/DB content dependencies.
Narration, caption, and hashtags are generated by Grok each run.

## Source Rules
Content principles are derived only from teachings associated with:
- Corey Wayne
- The Gottman Institute
- Esther Perel

Do not mention these names in generated narration/caption/hashtags.

## Prepare Narration + Scene Prompts (Page 4)
Run:
`pages/page4_relationship/prepare_page4_narration_and_scenes.cmd`

This does:
- Generates narration + caption + hashtags from Grok (JSON)
- Generates Speechma voice MP3
- Computes scene count using voice duration: `ceil(voice_seconds / 6)`
- Asks Grok CLI for strict JSON scene prompts
- Converts JSON to blank-line scene text
- Writes manifest with all output paths

## Run Grok Video Generation (CLI-only)
Run:
`python pages/automation_tools/grok_video_cli/grok_cli_scene_generate.py --prompt-file <scene_prompt_txt> --output-dir <output_folder>`

No UI fallback is used in this flow.
The script writes an event file:
`grok_outputs.done.json`

## Single-Pass Rendering (Event-Driven)
Run:
`pages/page4_relationship/scripts/render_page4_singlepass.cmd <manifest_json>`

Behavior:
- Waits for `grok_outputs.done.json` and expected scene files (from manifest scene_count).
- Concatenates all scene videos in order.
- Single-stage final render target: `720x1280`, `9:16`, with narration audio.
- Writes completion marker:
`page4_render_<item_id>.done.json`

## End-to-End Local Pipeline
Run:
`pages/page4_relationship/run_page4_full_pipeline.cmd`

This does:
1. Prepare narration/caption/hashtags + Speechma voice + scene prompt text.
2. Run Grok CLI video generation.
3. Run single-pass render after event marker appears.
